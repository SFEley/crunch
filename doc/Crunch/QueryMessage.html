Constructor Details

- (QueryMessage) initialize(sender, opts = {})

The only required parameter is a Crunch::Collection object. The query criteria, options, etc. will be happily accepted as options, or can be set via accessors at any time prior to sending. (Note: At this time, tailable cursors and other advanced query options are not supported.)

Parameters:

  • (Crunch::Document, Crunch::Group) sender —

    The object sending this message (so that we know where to route replies)

  • (optional Hash) opts (defaults to: {}) —

    Optional parameters

Options Hash (opts):

  • (Hash) :query — default: {} —

    The selection criteria

  • (Array) :fields — default: [] —

    An optional list of fields to return (returns all if empty)

  • (Integer) :skip —

    Skip the first x records (for paging)

  • (Integer) :limit —

    Return only y records

def initialize(sender, opts={})
  @sender = sender
  @collection_name = sender.collection_name
  @conditions  = opts[:conditions]  || {}
  @fields = opts[:fields] || []
  @skip   = opts[:skip]   || 0
  @limit  = opts[:limit]  || 0
end

def body
  conditions_bson = BSON.serialize @conditions      
  "\x00\x00\x00\x00#{collection_name}\x00#{[skip, limit].pack('VV')}#{conditions_bson}#{field_bson}"
end
